# 10) 화면 구성 (랜딩 없이 "서비스 화면"이 첫 화면)

## 개요

FORMTION의 모든 화면 구성과 레이아웃을 상세히 정의한다. 랜딩 페이지 없이 바로 서비스 화면으로 진입하는 구조이다.

---

## 10.1 Viewer 화면 (공유 링크 접속 시)

### 10.1.1 전체 레이아웃

#### 기본 구조
```
┌─────────────────────────────────────────┐
│  [상단 폼 영역] (옵션)                  │
├─────────────────────────────────────────┤
│                                         │
│  [Notion 콘텐츠 임베드]                 │
│  (iframe)                               │
│                                         │
│  [Inline Blur Gate 구간] (옵션)        │
│                                         │
├─────────────────────────────────────────┤
│  [하단 폼 영역] (옵션)                  │
└─────────────────────────────────────────┘
         [Floating CTA] (옵션, 우하단 고정)
         [서비스 로고 버튼] (우하단, 항시 표시)

[Entry Modal] (옵션, 오버레이 위)
```

---

### 10.1.2 상단 폼 영역 (Top Form)

#### 조건
- 운영자 설정에서 "상단 폼"이 ON인 경우만 표시

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [Title]                                │
│  이 템플릿, 바로 써도 됩니다.            │
│                                         │
│  [Sub-title]                            │
│  짧게 입력하면 블러가 즉시 풀립니다.     │
│                                         │
│  [이메일 입력 필드] *                   │
│  [이름 입력 필드]                       │
│  [회사명 입력 필드]                     │
│  [직무 선택 필드]                       │
│  [한 줄 질문 입력 필드]                 │
│                                         │
│  ☐ 개인정보 처리에 동의합니다. (필수)   │
│  ☐ 마케팅 수신 동의 (선택)              │
│                                         │
│  [Unlock & Continue 버튼]               │
│                                         │
│  No spam. One-time unlock.             │
└─────────────────────────────────────────┘
```

#### 스타일
- 배경: 화이트 (#FFFFFF) 또는 약간의 그레이 (#F7F7F8)
- 패딩: 상하 40px, 좌우 20px (모바일: 상하 30px, 좌우 16px)
- 테두리: 하단만 얇은 테두리 (#E9E9EC) 또는 그림자
- 최대 너비: 1200px (중앙 정렬)

#### Unlocked 상태
- 폼이 숨겨지거나
- "Unlocked" 배지만 표시 (운영자 설정)

---

### 10.1.3 Notion 콘텐츠 임베드 영역

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [Notion iframe]                        │
│  width: 100%                            │
│  height: auto (콘텐츠에 맞춰)           │
│  border: none                           │
│                                         │
│  [로딩 스피너] (초기 로딩 시)           │
│                                         │
│  [블러 오버레이] (블라인드 적용 시)     │
│                                         │
│  [Inline Blur Gate 카드] (옵션)        │
└─────────────────────────────────────────┘
```

#### 스타일
- 너비: 100%
- 높이: 콘텐츠에 맞춰 자동 조정
- 최대 너비: 1200px (중앙 정렬)
- 패딩: 좌우 20px (모바일: 좌우 16px)

---

### 10.1.4 Inline Blur Gate 구간

#### 조건
- 운영자 설정에서 "Inline Blur Gate"가 ON인 경우
- 블러 구간에 도달했을 때

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [정상 콘텐츠]                          │
│                                         │
├─────────────────────────────────────────┤
│  [블러 처리된 콘텐츠]                   │
│  (blur + overlay)                       │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │  [인라인 카드]                  │   │
│  │  Title: 여기부터 핵심입니다.    │   │
│  │  Sub-title: 이메일만 입력하면   │   │
│  │  [폼 필드들]                    │   │
│  │  [Unlock this section 버튼]     │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

#### 스타일
- 인라인 카드:
  - 배경: 화이트 (#FFFFFF)
  - 둥근 모서리: 16px
  - 그림자: 중간 강도
  - 패딩: 24px
  - 위치: 블러 구간 중앙 또는 상단

---

### 10.1.5 하단 폼 영역 (Bottom Form)

#### 조건
- 운영자 설정에서 "하단 폼"이 ON인 경우만 표시

#### 레이아웃
- 상단 폼과 동일한 구조
- 카피는 별도 설정 가능

---

### 10.1.6 Floating CTA 버튼

#### 조건
- 운영자 설정에서 "Floating CTA"가 ON인 경우

#### 레이아웃 (PC)
```
                    ┌────────┐
                    │ Unlock │  ← 우측 하단 고정
                    └────────┘
```

#### 레이아웃 (모바일)
```
┌────────────────────┐
│                    │
│                    │
│                    │
├────────────────────┤
│   [Unlock 버튼]    │  ← 하단 고정
└────────────────────┘
```

#### 스타일
- 배경: 오렌지 (#FF5A1F)
- 텍스트: 흰색 (#FFFFFF)
- 둥근 모서리: 상단만 둥글게 (북마크 느낌)
- 그림자: 최소한 (border 강조)
- 크기: PC 60x44px, 모바일 전체 너비 x 56px
- 위치: `position: fixed`, 우측 하단 20px (모바일: 하단 0px)

#### 클릭 시
- Drawer (모바일) 또는 Modal (PC) 오픈
- 폼 표시

---

### 10.1.7 서비스 로고 버튼

#### 조건
- 항시 표시 (운영자 설정 불가)

#### 레이아웃
```
                    ┌──────┐
                    │  F   │  ← 우측 하단, Floating CTA 위
                    └──────┘
```

#### 스타일
- 배경: 화이트 (#FFFFFF)
- 테두리: 얇은 테두리 (#E9E9EC)
- 둥근 모서리: 8px
- 크기: 44x44px
- 위치: `position: fixed`, 우측 하단 80px (Floating CTA 위)
- 클릭 시: FORMTION 홈페이지 또는 정보 페이지로 이동

---

### 10.1.8 Entry Modal

#### 조건
- 운영자 설정에서 "Entry 모달"이 ON인 경우
- 페이지 로드 완료 후 즉시 표시

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [오버레이: 반투명 검정]                 │
│                                         │
│        ┌──────────────────┐            │
│        │  [모달 박스]     │            │
│        │                  │            │
│        │  Title           │            │
│        │  이 콘텐츠는     │            │
│        │  체크인 후 열립니다.           │
│        │                  │            │
│        │  Body            │            │
│        │  입력 후 바로    │            │
│        │  전체 내용이 열립니다.         │
│        │                  │            │
│        │  [폼 필드들]     │            │
│        │                  │            │
│        │  [Unlock 버튼]   │            │
│        │  [Not now 버튼]  │            │
│        │  (닫기 허용 시)  │            │
│        └──────────────────┘            │
│                                         │
└─────────────────────────────────────────┘
```

#### 스타일
- 오버레이: `rgba(0, 0, 0, 0.5)`, 전체 화면
- 모달 박스:
  - 배경: 화이트 (#FFFFFF)
  - 둥근 모서리: 16px
  - 그림자: 강한 그림자
  - 최대 너비: 500px
  - 패딩: 32px
  - 위치: 화면 중앙 (모바일: 하단 또는 중앙)

## 10.2 Admin 화면 (운영자 콘솔)

### 10.2.1 프로젝트 리스트 화면

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [헤더]                                 │
│  내 프로젝트              [+ 새 프로젝트]│
├─────────────────────────────────────────┤
│  [검색] [필터] [정렬]                   │
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────┐   │
│  │ [프로젝트 카드]                 │   │
│  │ 프로젝트 이름                   │   │
│  │ notion.so/...                   │   │
│  │ formtion.app/p/abc123          │   │
│  │ 생성일: 2024-01-01              │   │
│  │ 리드: 10개                      │   │
│  │ [설정] [미리보기] [리드] [삭제] │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │ [프로젝트 카드 2]               │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

#### 프로젝트 카드
- 너비: 그리드 레이아웃 (PC: 3열, 모바일: 1열)
- 배경: 화이트 (#FFFFFF)
- 테두리: 얇은 테두리 (#E9E9EC)
- 둥근 모서리: 8px
- 패딩: 24px
- 호버: 그림자 효과

### 10.2.3 프로젝트 설정 화면

#### 레이아웃
```
┌─────────────────────────────────────────┐
│  [헤더]                                 │
│  프로젝트 설정         [저장] [미리보기]│
├──────────────┬──────────────────────────┤
│              │                          │
│  [탭 메뉴]   │  [설정 패널]            │
│  기본 설정   │                          │
│  폼 설정     │  Notion URL:            │
│  고급 설정   │  [입력 필드]            │
│  리드 설정   │                          │
│              │  UX 패턴 선택:          │
│              │  ☐ 상단 폼              │
│              │  ☐ 하단 폼              │

│              │                          │
│              │  [고급 설정 ▼]          │
│              │                          │
└──────────────┴──────────────────────────┘
```

#### 탭 구조
- **기본 설정**: Notion URL, UX 패턴, 블라인드 설정
- **폼 설정**: 필드 선택, 동의 옵션, 카피 설정
- **고급 설정**: 블러 강도, 키워드 블랙아웃, 저장 정책 등
- **리드 설정**: Webhook, CSV 내보내기 등

## 10.3 반응형 디자인

### 10.3.1 브레이크포인트
- 모바일: < 768px
- 태블릿: 768px - 1024px
- PC: >= 1024px

### 10.3.2 모바일 최적화
- 터치 친화적인 버튼 크기 (최소 44x44px)
- 전체 너비 사용
- Drawer 형태의 모달 (하단에서 슬라이드 업)
- 스와이프 제스처 지원

### 10.3.3 PC 최적화
- 최대 너비 제한 (1200px)
- 중앙 정렬
- 호버 효과
- 키보드 단축키 (추후 추가)

---

## 10.4 접근성

### 10.4.1 키보드 네비게이션
- Tab 키로 모든 인터랙티브 요소 이동
- Enter 키로 버튼 클릭
- Esc 키로 모달 닫기

### 10.4.2 스크린 리더
- 모든 폼 필드에 적절한 라벨
- ARIA 라벨 제공
- 에러 메시지를 스크린 리더가 읽을 수 있도록

### 10.4.3 색상 대비
- WCAG AA 기준 준수 (최소 4.5:1 대비율)
- 색상만으로 정보를 전달하지 않음 (아이콘/텍스트 병행)
